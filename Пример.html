Контрольные вопросы:

1. Что такое Flexbox?

CSS свойство, которое позволяет легко выравнивать и распределять пространство между элементами в контейнере, даже когда их размер неизвестен или динамичен.

2. Какие основные свойства флекс-контейнера?

display: flex, flex-direction, justify-content, align-items, flex-wrap, align-content.

3. Каковы основные свойства флекс-элементов?

order, flex-grow, flex-shrink, flex-basis, flex, align-self.

4. Как можно выровнять элементы по центру по горизонтали и/или по вертикали с помощью Flexbox?

По центру по горизонтали: justify-content: center. По вертикали: align-items: center. Для центрирования по центру justify-content: center и align-items: center.

5. Что делает свойство flex-wrap?

Определяет, должны ли флекс-элементы переноситься на новую строку, если они не помещаются в одну строку.

6. Как растянуть три флекс элемента в отношении 1:2:1?

Для этого нужно задать элементам свойство использовать сокращенное свойство flex: 1(первый элемент), flex: 2(второму), flex: 1(третьему).
